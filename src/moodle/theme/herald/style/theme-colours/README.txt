Theme Colour Options
=====================

Herald has 4 colour schemes. If you don't like the default colour scheme you can easily switch to a different colour scheme by following the steps below:

1) Use Blue Colour Scheme
Replace /herald/style/style.css with /herald/style/theme-colours/blue/style.css

2) Use Green Colour Scheme
Replace /herald/style/style.css with /herald/style/theme-colours/green/style.css

3) Use Purple Colour Scheme
Replace /herald/style/style.css with /herald/style/theme-colours/purple/style.css
